Use Cases

Capacity Analysis

FREEVAL provides freeway planning-level capacity analyses based on HCM6 for undersaturated and oversaturated conditions.

Work Zone Impact Assessment Analysis

FREEVAL provides work zone impact assessments based on HCM6 for planning-level and operational-level analyses.

Managed Lanes Analysis

FREEVAL evaluates the operation of managed lanes such as (HOV, HOT, Express ways) based on HCM6.

Travel Time Reliability Analysis

FREEVAL assesses the travel time reliability of freeway systems based on the HCM6 reliability scenario generation method.

FREEVAL-NC Freeway Segmentation Database

The FREEVAL-NC Freeway Segmentation Database contains geometric and demand information for North Carolina freeways, interstate and non-interstate, which can be imported into the FREEVAL-NC computational engine. It contains the following data:

  • Geometry (segment type, segment length, free flow speed, number of lanes, etc.)
  • Google Maps Integration
  • Demand (flow rates, percent trucks, etc.)
Go to the FREEVAL-NC Freeway Segmentation Database

FAQ

How do I download FREEVAL-WZ or FREEVAL-DSS?

FREEVAL+ contains all FREEVAL-WZ and FREEVAL-DSS modules. You can download FREEVAL+ from this website. Once you execute the program, go to "Toolbox Menu" and click on FREEVAL-WZ or FREEVAL-DSS.

What are the system requirements for FREEVAL?

FREEVAL is written in Java and, therefore, only requires the Java SE Runtime Environment 8 (JRE). Consult the system requirements for the JRE here.

How do FREEVALâ€™s analyses compare to those specified in the Highway Capacity Manual (6th ed.)?

FREEVAL replicates the methods of analysis specified in Chapters 10, 11, 12, 13, 14, 25, and 26 of the Highway Capacity Manual (6th ed.).
