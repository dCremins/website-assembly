FAQ

Remember! If you cannot find your question here, send it to support@freeval.org

What are the Operating System Requirements for FREEVAL?

FREEVAL is developed in Java environment and as such it can be run under any operating systems ranging from MS Windoes and Mac OS to Linux operating system. The only requirement is to install Java Runtime Environment (JRE) version 8 and later.

How Similar or Different is FREEVAL Compared to HCM?

FREEVAL replicates the freeway analysis in Highway Capacity Manual (HCM). All methods documented in Chapters 10, 11, 12, 13, 14, 25, and 26 are implemented in FREEVAL. This includes freeway facilities, reliailty, managed lanes, work zones and etc. analyses.

What do I need to do if I have Questions or Problems Using FREEVAL?

If you have any problems using FREEVAL, please do not hesitate to contact us at support@freeval.org.

What would it take for me to customize FREEVAL to my own specific use cases?

We do customize FREEVAL to match your specific needs. This can range from connecting FREEVAL to in-house data sources to standard reports generations. please do not hesitate to contact us at support@freeval.org.
