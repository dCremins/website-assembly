Driver and Transportation Analytics Lab

D.a.T.A. Lab at ITRE provides analytical facilities to analyze big data, visualization, and predictions in transportation industry. This page serves as a central point to connect to different tools developed in ITRE's DATA Lab. For all tools developed in DaTA lab, a link is placed in this screen. Different tools will have a different authentication method. If you need access to these tools, please contact Behzad Aghdashi at saghdas@ncsu.edu.
