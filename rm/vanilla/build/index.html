I-540WB OnRamp Signal Evaluation Dashboard

This dashboard is designed to enable NCDOT to evaluate the impact of on-ramp signals on I-540WB in Raleigh, NC. Travel time data source for this dashboard is HERE.com and the volume data is collected using NCDOT fixed sensors.
